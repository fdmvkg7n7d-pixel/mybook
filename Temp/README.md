# mybook

This repository contains working notes, character profiles, scene outlines, timelines, and vacation/travel planning documents for an intergenerational fiction saga centered on the Key family.

## Start here

- [`Temp/TIMELINE.md`](Temp/TIMELINE.md) — master chronology and anchor dates.
- [`Temp/OPEN ITEMS.md`](Temp/OPEN%20ITEMS.md) — unresolved story questions, grouped by priority.
- [`Temp/EDITORIAL_AUDIT.md`](Temp/EDITORIAL_AUDIT.md) — current editing pass notes, structural issues, and recommended next steps.

## Editorial conventions

- Keep one clear `# H1` title at the top of each Markdown file.
- Use `##` and `###` for internal structure rather than repeated `#` headings unless a file is intentionally a combined anthology.
- Preserve canonical names and dates unless the master timeline or open-items file is updated at the same time.
- When resolving an open question, mark the checkbox in `Temp/OPEN ITEMS.md` instead of deleting the question, so the decision history remains visible.
- Some arcs include mature or traumatic material; line edits should prioritize clarity, consent/timeline consistency, and non-exploitative framing.
