# Family Map

A consolidated, cross-verified genealogy of the Key family saga, derived from `Temp/TIMELINE.md`, `Temp/MASTER_CHARACTER_INDEX.md`, and each branch's master/profile files. Open any `.html` file in a browser (or the workspace preview); all files are fully self-contained — no external resources.

## Files

| File | What it is |
|---|---|
| [`FAMILY_TREE.html`](FAMILY_TREE.html) | **Master tree** — one connector-line diagram: Gen 1 → the eight sisters → their children → the grandchildren, with relationship-type badges (bio / adopted / foster / step / surrogate-donor / guardianship / twin-triplet). |
| [`FAMILY_BRANCHES.html`](FAMILY_BRANCHES.html) | **Per-branch view** — each of the eight sister-branches as its own zoomable panel, with a jump-nav index, meta tags, and mini-timelines for the defining arcs (Carlota's kidnapping, Brandi's cancer). |
| [`FAMILY_TIMELINE.html`](FAMILY_TIMELINE.html) | **Date-driven ribbon** — ~48 canonical events plotted on a linear year axis (2020–2096), colour-coded by generation, with working category filters. |
| [`FAMILY_TREE.md`](FAMILY_TREE.md) | The same genealogy as clean, editable Markdown (tables + indented tree + external-allies list + cross-family motif notes). |

## Colour key (shared across all three views)

| Colour | Generation |
|---|---|
| indigo | Gen 1 — Mike Key & Terri Anne |
| rose | Gen 2 — the eight sisters |
| teal | Gen 3 — the children |

Timeline-only markers: violet = pivotal event · slate diamond = death / loss.

## Relationship badges

`BIO` · `ADOPTED` · `FOSTER` · `STEP/SOCIAL` · `SURROGATE/DONOR` · `GUARDIANSHIP` · `TWIN/TRIPLET`

## Notes

- Birth years marked **~** are canonical estimates from the source; exact placement uses the timeline's anchor years.
- Open/TBD items (exact calendar dates, several surnames, some partner backgrounds) are preserved as the source left them — nothing was invented.
- If the saga evolves, update `Temp/TIMELINE.md` / the branch master files first, then regenerate or hand-edit these views to match.
